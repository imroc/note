# 创建网站

一般都会用 classic 这个 preset，使用 npx 初始化网站：

```bash
npx create-docusaurus@latest imroc.cc classic
```

如果想用比较新的版本，可以指定 next 版本:

```bash
npx create-docusaurus@next imroc.cc classic
```
